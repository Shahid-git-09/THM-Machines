## Methodology

Every writeup follows a structured approach:
1. **Enumeration** — Nmap, service fingerprinting
2. **Vulnerability Identification** — CVE research, NSE scripts
3. **Exploitation** — Manual where possible, Metasploit as fallback
4. **Post Exploitation** — Privilege escalation, flag retrieval
5. **Key Learnings** — What the machine taught, what to improve
